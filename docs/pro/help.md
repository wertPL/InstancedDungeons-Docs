# Help Center

Use this section when something needs checking after setup.

## Recommended Order

1. Run validation:

```text
/dungeon validate <id>
```

2. Check the troubleshooting page.
3. Check the changelog for changed behavior.
4. Join Discord if the issue still looks like a plugin bug.
